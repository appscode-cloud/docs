# Customize Cluster Sidebar

The Cluster Details page in AppsCode Dashboard features a dedicated sidebar that displays standard Kubernetes workloads and various AppsCode custom resources by default. This sidebar provides quick access to resource details pages and related features. However, AppsCode recognizes the importance of customization to tailor the user experience.

## Cluster Sidebar Settings

AppsCode offers a Cluster Sidebar Settings page, allowing users to personalize their sidebar preferences. The settings page comprises two panels:

1. **Available Menus (Left Side):** Displays all available menu options that users can choose from. Users can utilize the search functionality to find specific resources efficiently.

2. **Current Menus (Right Side):** Reflects the current menu options visible in the sidebar.

## Customization Process

To customize the cluster sidebar, follow these steps:

1. **Drag and Drop:** Users can easily move a menu option from the available options to the current menu section and vice versa. This drag-and-drop functionality simplifies the customization process.

2. **Search for Resources:** Utilize the search bar in the available menus section to quickly locate specific resources. Once found, users can drag the resource to the current menu section.

3. **Default Options:** Users have the option to revert to the default sidebar options by clicking the "Default" button. This action undoes any previous customization.

4. **Save Changes:** To apply the personalized changes, users need to click the "Save" button. This ensures that the sidebar reflects the selected menu options.

This enhanced customization feature empowers users to efficiently organize and optimize their AppsCode Dashboard experience by arranging the sidebar according to their preferences.
